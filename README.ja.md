# showacam
日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

ブラウザ上で使えるシンプルなカメラアプリケーション。

## デモ
[https://code4fukui.github.io/showacam/](https://code4fukui.github.io/showacam/)

## 特徴
- デバイスのカメラから画像をキャプチャできます
- キャプチャした画像を共有するためのQRコードを生成できます
- 前面カメラと背面カメラをサポートしています

## 使用法
1. ウェブブラウザでアプリケーションを開きます。
2. "巻く"ボタンをクリックしてカメラを起動します。
3. "撮る"ボタンをクリックして画像を撮影します。
4. "QR"ボタンをクリックして撮影した画像のQRコードを生成します。
